# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.12.x   | :white_check_mark: |

## Reporting a Vulnerability

If a vulnerability or security concern is found, you can report it directly via the support email: suporte@opensourcecommunitybrasil.com.br.

Make sure to mention the PascalRAL project in the subject so we can adress it as quickly as possible.
