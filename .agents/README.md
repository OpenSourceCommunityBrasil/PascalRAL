# .agents

Pasta para materiais auxiliares de automação e agentes de IA.

- `AGENT_QUICKSTART.md` — guia mínimo (baixo consumo de contexto) para escolher o próximo arquivo-alvo.
- `PROJECT_MAP.md` — mapa amigável do repositório para acelerar entendimento e navegação por agentes de IA.
- `SKILLS.md` — instruções e guidelines para melhorar desempenho/eficiência dos agentes.

> Use primeiro `AGENT_QUICKSTART.md` quando a janela de contexto for pequena.
